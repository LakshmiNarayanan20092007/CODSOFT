# 🎬 AI Movie Genre Classifier

A professional Machine Learning web application that predicts movie genres using NLP and Logistic Regression.

---

## 🚀 Features

- Predicts movie genres from descriptions
- Beautiful modern UI
- Flask web application
- NLP text processing
- TF-IDF Vectorization
- Logistic Regression model

---

## 🛠 Technologies Used

- Python
- Flask
- Scikit-learn
- HTML
- CSS

---

## 📂 Dataset

Genre Classification Dataset provided for the internship project.
