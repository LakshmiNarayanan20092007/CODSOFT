# AI Spam SMS Detector

A professional Machine Learning web application that detects whether an SMS message is Spam or Not Spam using Natural Language Processing (NLP).

## Features
- Detects spam SMS messages
- Machine Learning based prediction
- Modern glassmorphism UI
- Built using Flask
- Real-time prediction system
- Responsive and professional design

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes Algorithm
